Introduction to CSS

What is CSS?

CSS stands for Cascading Style Sheets. This is how we style the html in a webpage to make it look better rather than the basic white page with black text in Times New Roman and all of the elements being left alligned. This is usually made on a separate file so we don't mix up the CSS with other code, and it makes it easier to look through.

Types of CSS

There are a 4 different types of CSS

  1. Inline Styles
    • It is configured in the body of the webpage
    • It uses the <style> attribute of the HTML tag it is in
    • Only applies the style to the tag it is found in
  2. Embedded Styles
    • This is configures in the head of the HTML file
    • It uses the HTML <style> element
    • This applies the style to the entire webpage
  3. Imported Styles
    • It is very similar to external style sheets
    • We will look at this in future but will mainly focus on external
  4. External Styles
    • This is configured in a separate file in ending with .css
    • To then apply the styles to the main document we must use the HTML element <link>
    • This is the only type of styling we will be using

What are its advantages?

  • It gives the user a greater control on page layout and typography
  • It separates the style and the structure of the webpage so the code is more organized
  • The styles can be stored in a file and linked to separate documents, but you can link it to many different documents
  • Because we store the styles on a separate page it's easier to do maintenence on the webpage later on
  • It reduces the size of our documents by not having all the styles inline with the html
